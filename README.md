# DT207G – Kursdatabas

En webbapplikation för att lagra och visa kurser med hjälp av Node.js, Express och MySQL.

## Funktioner

- Visa kurser från databas
- Lägg till nya kurser via formulär
- Radera kurser
- Server-side validering
- Responsiv design

## Tekniker

- Node.js
- Express
- MySQL
- EJS 
- HTML, CSS

## Kom igång

1. Klona repo
2. Skapa databas:
    ```bash
    mysql -u root < database.sql
    ```
3. Kör applikationen:
    ```bash
    node server.js
    ```
