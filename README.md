# DT207G – Kursdatabas

En webbapplikation för att lagra och visa kurser med hjälp av Node.js, Express och SQLite.

## Funktioner

- Visa kurser från databas
- Lägg till nya kurser via formulär
- Radera kurser
- Server-side validering
- Responsiv design

## Tekniker

- Node.js
- Express
- EJS 
- HTML, CSS
